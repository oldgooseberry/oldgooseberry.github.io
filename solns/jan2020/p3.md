# Question

{% raw %}{::nomarkdown}<div>$\text{log}_{10}(x) = 2 - \text{log}_{10}(25)$</div>{:/}{% endraw %}

# Solution

{% raw %}{::nomarkdown}<div>
$\begin{align*}
\log_{10}(x) &= 2 - \log_{10}25 \\
\log_{10}(x) + \log_{10}25 &= 2 \\
\end{align*}$
</div>{:/}{% endraw %}

but

$log_{b}(xy) = \log_{b}(x) + \log_{b}(y)$

thus

$\log_{10}(25 x) = 2$

however

$\log_{b} = c \Rightarrow  b^{c} = x$

therefore

{% raw %}{::nomarkdown}<div>
$\begin{align*}
25x &= 10^{2} = 100 \\
x &= 4
\end{align*}$
</div>{:/}{% endraw %}
